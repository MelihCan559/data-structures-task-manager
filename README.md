# Task Manager Using Data Structures (C++)

This project demonstrates the implementation of core data structures such as linked lists, stacks, and queues in C++.

## Features
- Add tasks using linked list
- Process tasks in queue order
- Delete last task using stack logic
- Undo delete functionality

## Technologies
- C++
- Pointers
- Dynamic memory allocation

## Purpose
This project was built to strengthen understanding of data structures and low-level memory management.
